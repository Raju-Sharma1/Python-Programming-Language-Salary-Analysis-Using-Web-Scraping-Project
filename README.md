# Python-Programming-Language-Salary-Analysis-Using-Web-Scraping-Project
This project demonstrates web scraping and data extraction techniques using Python. The objective was to extract programming language salary data from an HTML webpage, process the information, and generate a structured CSV dataset for further analysis.

# Objective
To scrape programming language salary information from an online HTML table and convert the unstructured web data into a clean, structured dataset.

# Tools & Technologies Used
1. Python
2. Requests (HTTP requests)
3. BeautifulSoup (Web Scraping)
4. Pandas (Data Processing & Export)
5. NumPy

# Project Workflow
1. Sent an HTTP request to retrieve webpage content
2. Parsed HTML using BeautifulSoup
3. Located and extracted table data
4. Extracted programming language names and average salary values
5. Cleaned and structured data using Pandas
6. Removed unwanted header rows
7. Exported processed data into a CSV file
